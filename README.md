## Readable App
*Inorder to start the readable app first start api server with below steps*
* Install and start the API server
    - `cd api-server`
    - `npm install`
    - `node server`
* In another terminal window, run front-end
    - `cd frontend`
    - `npm install`
    - `npm start`

Go to `localhost:3000` to access the app
